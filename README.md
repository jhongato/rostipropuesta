# Propuesta de Automatización Inteligente: La Rostisería

Este repositorio aloja la landing page de la propuesta comercial para implementar un ecosistema de agentes inteligentes en **La Rostisería**.

El objetivo de este proyecto estático es presentar dos opciones de digitalización del servicio al cliente vía WhatsApp.

## 🌟 La Propuesta

La propuesta se divide en dos fases/planes:

1. **Estado Actual: El Prototipo (Fase 1)**
   Un Especialista en Ventas cuyo objetivo único es resolver dudas sobre la carta (platos, ingredientes, precios) y generar deseo de compra para redirigir al cliente al menú digital.

2. **Opción 1: Plan "Esencial" (Optimización del Prototipo)**
   Ideal para digitalizar la atención básica sin complicaciones. Incluye atención 24/7 sobre el menú y restaurante con soporte para 1 línea de WhatsApp. Redirige estratégicamente los pedidos.

3. **Opción 2: Plan "Ecosistema Multi-Agente" (Máximo Valor)**
   Transforma el WhatsApp en un centro de operaciones inteligente. Incluye agentes delegados de Ventas + Upselling, Cross-Selling, Reservas guiadas por calendario y Atención al Cliente (PQR). Adicionalmente integra un Dashboard de métricas.

## 🚀 Despliegue en GitHub Pages

Este proyecto consta de un único archivo `index.html` estructurado visualmente con Tailwind CSS. Está configurado para ser desplegado fácilmente con GitHub Pages.

**Para habilitarlo en su repositorio:**
1. Ve a `Settings` en tu repositorio de GitHub.
2. Navega a `Pages` en el panel izquierdo.
3. Bajo `Build and deployment`, selecciona "Deploy from a branch".
4. Selecciona la rama `main` (o la que estés utilizando) y la carpeta `/ (root)`.
5. Haz clic en **Save** y espera unos minutos a que GitHub procese y habilite tu página.
